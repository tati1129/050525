Написать приложение для отдела кадров, которое должно
реализовывать следующий функционал:

1) зарегистрировать соискателя на должность
2) принять на работу и добавить в общий список сотрудников
3) из списка сотрудников распределить по отделам

-------------------
Person
- имя
- специальность
...

Соискатель (Applicant)
- int id
- Person person
- boolean isApproved

Сотрудник
- int id
- Person person
- департамент
- должность
- зарплата

Департамент:
- название
- общее количество струдников

Соискатели
- Applicant[] applicants

Сотрудники
- Employee[] employees

Департаменты
- Department[] departments


Компания:
- соискатели
- сотрудники
- департаменты
