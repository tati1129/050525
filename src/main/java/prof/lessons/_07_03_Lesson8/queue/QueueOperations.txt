

offer(element) - вставить новый элемент в хвост очереди
add(element) - вставить элемент в хвост

Отличие в том, что эти методы возвращают в качестве результата boolean
и если элемент успешно добавлен, то true

Но если в процессе добавления добавить не получилось, то offer просто возвратит
false, а метод add() - вернет ошибку


Извлечение головного элемента (с удалением его из коллекции) :

- remove() - извлекает головной элемент, если при попытке извлечения выяснится, что очередь пустая
то вернет ошибку NoSuchElementException

- poll() - извлекает головной элемент, если очередь пуста, то возвращает null

Посмотреть головной элемент (без удаления из коллекции):

- element() - возвращает головной элемент, если пустая коллекция то NoSuchElementException
- peek() - возвращает головной элемент, если пустая коллекция то null
