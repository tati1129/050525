Лямбда оператор: ->

Трактуется как "становится" / "переходит"

код лямбда выражения может формироваться одним из двух способов:
1) одиночное выражение (одиночная лямбда)
2) блочное выражение - набор строк кода, заключенных в {}

Это тот случай, когда необходимо в правой части лямбда выражения указать
несколько операторов и обязательно указать return

(list_of_parameters) -> {
// тело лямбды
// ...
//...
return result;
}


list_of_parameters - список параметров, которые получает лямбда

Если в лямбда выражении содержится единственный параметр, то можно без ()

i -> {
//..
//..
//..
return result
}

Возможно, что лямбда НЕ ПОЛУЧАЕТ никаких параметров
() -> {
//..
//..
//..
return result
}
